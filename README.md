# Project: Personal Goals

This project aims to enhance personal development within the workplace. It includes a real-time game and follows a monorepo structure using TurboRepo.

- Technology Stack:
  - Server:
    - Node.js
    - Express
    - Socket.io
    - TypeScript
  - Client:
    - Vue 3
    - TypeScript

## Install package

pnpm i

## Start both sever and client

pnpm dev
